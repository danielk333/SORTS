Profiler¶

class sorts.profiling.Profiler(distribution=False)[source]¶

Bases: object

Performance profiler class.

The named performance control is especially useful for timing contents of loops.

Usage example:

p = Profiler()
p.start('list init')
p.start('program')
lst = list(range(200))
p.stop('list init')
for i in range(1000):
    p.start('list reversal')
    lst = lst[::-1]
    p.stop('list reversal')
p.stop('program')

print(p)
