SGP4_module_wrapper.__init__(mjd_epoch, mean_elements, B)[source]¶

Initialize SGP4 object from mean orbital elements and ballistic coefficient

Creates a sgp4.model.Satellite object for mean element propagation First all units are converted to the ones used for TLEs, then they are modified to the sgp4 module standard.

mjd_epoch : epoch as Modified Julian Date (MJD) mean_elements : [a0,e0,i0,raan0,aop0,M0] B : Ballistic coefficient ( 0.5*C_D*A/m )

This object is not usable for TLE generation, but only for propagation as internal variables are modified by sgp4init.
